# Fitness Tracker Bot

Телеграм-бот для отслеживания дневной нормы воды, калорий и физической активности. Бот рассчитывает индивидуальные нормы на основе параметров тела и текущей погоды.

## Функционал

* Регистрация профиля: Расчет норм воды и калорий на основе веса, роста, возраста и уровня активности.
* Учет погоды: Автоматическое увеличение нормы воды, если в указанном городе жарко.
* Трекинг питания: Ввод еды текстом, например, "борщ" или "яблоко".
* Трекинг воды: Логирование выпитой воды и сожженных калорий.
* Визуализация: Построение графиков прогресса за день.

## Стек технологий

* Python 3.11+
* **aiogram 3.24
* SQLAlchemy
* Matplotlib
* Aiohttp
* Docker

## Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/krasnovdanial/Fit_Bot.git
    cd Fit_Bot
    ```

2.  Установите зависимости и окружение командой:
    ```bash
    uv sync
    ```

3.  Создайте файл `.env` в корне проекта и заполните его:
    ```text
    TOKEN=...
    WEATHER_KEY=...
    ```

4.  Запустите бота:
    ```bash
    uv run main.py
    ```
